## 📦 Installation

```bash
# npm
$ npm install i18n-json-translate

# yarn
$ yarn add i18n-json-translate

# Usage
$ json-translate
```

## 🌈 Usage Scenery

For use in the following scenarios：

> Use the translation result of key as its value

```json
{
  "你好": "",
  "苹果": ""
}
```
