## 📦 下载

```bash
# npm
$ npm install i18n-json-translate

# yarn
$ yarn add i18n-json-translate

# Usage
$ json-translate
```

## 🌈 使用场景

针对于以下场景进行使用：

> 将 key 的翻译结果作为其 value

```json
{
  "你好": "",
  "苹果": ""
}
```

## ⭐ 集成支持

| 厂商       | 使用条件                               |
| ---------- | -------------------------------------- |
| 百度翻译   | 需注册百度翻译账号，并依据开发者文档   |
| 有道云翻译 | 需注册有道云翻译账号，并依据开发者文档 |
| ChatGPT    | 需提供 API Key                         |
